# AI-Powered News Classification App

## Overview
This project is a Streamlit web app that classifies Sri Lankan news articles into categories using a fine-tuned Hugging Face Transformer model.  
It also provides interactive features like sentiment analysis, readability scoring, article summaries, and a Q&A interface, allowing users to explore news content intelligently.

## Features
- Classifies news articles into multiple categories  
- Sentiment analysis and readability metrics for each article  
- Summarizes news articles for quick insights  
- Interactive Q&A functionality using the NLP model  
- User-friendly Streamlit web interface  

## Dataset
- News articles collected from publicly available Sri Lankan news sources  
- Preprocessed and cleaned for NLP classification  

## Tools Used
- Python  
- Hugging Face Transformers (for NLP classification)  
- Streamlit (for web interface)  
- Pandas, NumPy (data handling)  
- Matplotlib, Seaborn (visualizations)
